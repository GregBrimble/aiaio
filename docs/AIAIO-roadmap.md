## AIAIO Roadmap

This template may, at some point in the future, be updated to support the following:

- Web application
  - Ability to set HTTP status from an `@aviation/router` route
  - Ability to set HTTP headers from a `@aviation/router` route
  - Ability to SSG (static site generation) a set of routes
  - Support for the "use cache" directive
  - Automatic re-building of `./lib/` packages in Vite
